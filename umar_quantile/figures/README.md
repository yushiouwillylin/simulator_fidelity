# UMAR Figures

This directory stores archived PNG copies of the UMAR reproduction figures.

Expected main files:
- `umar_main_variance_adjusted_upper_curves.png`
- `umar_simulator_rmse.png`

Figure roles:
- `umar_main_variance_adjusted_upper_curves.png`: the paper-facing fidelity plot. It summarizes the calibrated upper quantile curve for scenario-level variance discrepancy across emulator families.
- `umar_simulator_rmse.png`: a construction-stage diagnostic for out-of-fold predictive accuracy. It is useful context, but it is not the main fidelity object of the method.
